# 📦 Automated Backup & Recovery System


## Project Overview

Automated Backup & Recovery System is a Python-based application designed to create, manage, restore and monitor data backups.

The system provides automatic backup scheduling, recovery functionality, logging, reporting and security features.



## Features

### Backup Management

- Create manual backups
- Automatic scheduled backups
- Backup history tracking
- Backup size calculation


### Recovery System

- Restore previous backups
- Select backup versions
- Recover data safely


### Monitoring

- System activity logging
- Error tracking
- Database logs


### Reporting

- Excel backup reports
- PDF backup reports
- Download reports


### Security

- Backup encryption
- Secure key management


## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- OpenPyXL
- ReportLab
- APScheduler
- Cryptography
